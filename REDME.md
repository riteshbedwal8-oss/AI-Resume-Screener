# 🚀 AI Resume Screening System

An AI-powered Resume Screening and Candidate Ranking System built using Python, NLP, and Streamlit.

---

## ✨ Features

- Resume Parsing (PDF/DOCX)
- AI-Based Resume Matching
- NLP Semantic Similarity
- Skill Match Analysis
- Candidate Ranking
- CSV Export
- Modern Streamlit Dashboard

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Sentence Transformers
- spaCy
- Pandas
- Plotly
