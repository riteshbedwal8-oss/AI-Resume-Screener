# 🚀 AI Resume Screening System

Developed an AI-powered Resume Screening and Candidate Ranking System using Python, NLP, and Streamlit to automate the recruitment shortlisting process.

The system intelligently analyzes resumes against job descriptions using semantic similarity, skill extraction, and AI-based matching techniques. It ranks candidates based on relevance scores and identifies matched and missing skills to assist recruiters in faster and more accurate hiring decisions.

✨ Key Features:

* Resume Parsing for PDF and DOCX files
* AI-based Semantic Resume Matching
* NLP Skill Extraction and Gap Analysis
* Candidate Ranking Dashboard
* Interactive Modern Streamlit UI
* CSV Export Functionality
* Real-time Resume Processing

🛠️ Technologies Used:
Python, Streamlit, Sentence Transformers, spaCy, Pandas, NLP, Machine Learning
